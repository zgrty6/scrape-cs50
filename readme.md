## Requirements
- requests
```
pip install requests
```
- selenium
```
pip install selenium
```
- tqdm
```
pip install tqdm
```

## Usage
1. Run script.
```
python scrape-cs50.py
```
2. Copy/Paste url and it will download files in the same folder as the script.
